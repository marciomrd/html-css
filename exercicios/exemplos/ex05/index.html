Título gerado usando a tag <h1>!

Neste exercício estamos utilizando as diversas tags aprendidadas durante o curso. Para que o curso seja realmente eficaz temos que exercitar todas as técnicas aprendiadas períodicamente, assim como esse paragrafo que está utilizando a tag <p>. Essa tag é utilizada para inserir conteúdos de texto em htlm5, sendo que esses conteúdos ficam organizados em parágrafos como este que você está lendo agora.

Também estamos utilizando a tag <title> e <h1> que são tags de título, mas não se engane, a primeira serve apenas para os títulos que ficam na tag <head> do documento, ele vai indicar o título da página no navegador e aparece logo após o ícone de favoritos da página.

Também estamos utilizando a tag <br> para apresentar uma quebra de texto dentro deste mesmo parágrafo, além de muitos outros recursos utilizados como o ícones de favorito a seguir, que também está localizado ao topo da página.

Aqui será inserida uma imagem de paisagem usando a tag <img>!
